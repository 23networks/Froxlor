		assign (line 139)
	
 
	
	

Add one or more parameter to our request


If first parameter is a string, we'll use the second parameter to add a  single parameter to our request, if it's an array, we'll ignore the second  parameter and add all key/value-pairs of the array to our request
Example:  
  1.  $request->assign("command", "CheckDomains");
    2. 

  2.  $request->assign(array("domain0" => "example.org", "domain1" => "example.com"));
    3. 


	
    
				
  • return: Always true
    • 
				
  • access: public
